# Clear the outdated metadata in the default cache directory.

Clear the outdated metadata in the default cache directory.

## Usage

``` r
clear_old_metadata(updated_data)
```

## Source

[Mangiola et
al.,2023](https://www.biorxiv.org/content/10.1101/2023.06.08.542671v3)

## Arguments

- updated_data:

  A character vector of outdated metadata name

## Value

`NULL`, invisibly
